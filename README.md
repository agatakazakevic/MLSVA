# MLSVA
MLSVA pipeline for TSP Shapley-value approximation. Generates synthetic routing instances, extracts features, and benchmarks neural nets, SVR, KNN, RF, and XGBoost against Exact/MC/HMC labels. Includes cached datasets, MLSVA-style NN tuning, rich visualizations, and comparison tables in a single reproducible notebook.
